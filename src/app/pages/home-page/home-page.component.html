Welcome to my dashboard !

This dashboard contains :

  • A todo-list linked to a Mongo DB
  • Support for Spotify's API

With the todo list, you can :

  • Create a todo
  • Edit a todo
  • Delete a todo
  • Save the todo list

With Spotify, you can perform some actions when logging in, like :

  • Search an artist, an album or a track
  • Play any of these three (try it out !)
  • Volume & Play / pause controls
  • And many more features to come...

With Gmail, you can perform some actions when logging in, like :

  • Check your email list
  • Check your trash
  • Mark emails as important
  • Delete one or several emails
  • Put one or more emails in the trash
